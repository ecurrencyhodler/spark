# Bare on Expo

Example of embedding Bare in an Expo application using <https://github.com/holepunchto/react-native-bare-kit>.

## Usage

Start by installing the dependencies:

```sh
npm install
```

When finished, you can run the app on either iOS or Android.

### iOS

```sh
npm run ios
```

### Android

```sh
npm run android
```
